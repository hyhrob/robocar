这是一个儿童教育机器人编程平台。

# purpose
儿童机器人编程

# 配件表

- 树莓派3b，用于执行小车控制逻辑程序
- 两个电机驱动板（每个驱动板能驱动两个电机，或一个驱动板带四路驱动），用于执行逻辑并驱动直流电机
- 四个直流电机与电机驱动板连接，执行行走指令
- 后面可能增加：
  - SLAM室内导航
  - 执行逻辑解释器
